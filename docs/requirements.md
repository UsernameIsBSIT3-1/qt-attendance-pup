# Requirements (Draft v0.1)

### Functional Requirements
- Generate unique QR codes for students
- Scan QR codes using camera or uploaded image
- Record attendance in a database
- Display daily/weekly/monthly attendance reports
- Admin CRUD for users

### Non-Functional Requirements
- Easy-to-use interface
- Fast QR detection (target < 2 seconds)
- Secure handling of student data
- Scalability for multiple sections
