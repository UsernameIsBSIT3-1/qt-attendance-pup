# QR Code-Based Attendance System (Prototype) - Group 10

Minimal web-based prototype for II1G-17.

## What this includes
- **Frontend**: `frontend/` static pages (dashboard, scan, reports)
- **Backend**: `backend/` Express API (stores data to `data/db.json` via lowdb)
- Easy runnable prototype for demos.

## Run locally
1. Install Node.js (v16+ recommended)
2. From project root:
   ```bash
   npm install
   npm start
